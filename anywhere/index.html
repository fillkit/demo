Browser Extension

FillKit ships browser extensions for Chrome (Manifest V3) and Firefox, bringing context-aware form autofill to every website you visit. The extensions share the same core engine as the SDK but run entirely inside the browser — no server calls required for basic usage.

Features

  • 1
    One-click autofill — Click the extension icon or press the keyboard shortcut and every visible form is filled instantly.
  • 2
    Works on any website — No integration needed. The extension detects forms, identifies field types, and generates realistic data on the fly.
  • 3
    Locale & mode switching — Toggle between 50+ locales and switch between valid or invalid fill modes directly from the popup.
  • 4
    No account required — Basic autofill works entirely offline. Sign in with a FillKit token to unlock cloud datasets and team profiles.

How It Works

1

Install

Add the extension from the Chrome Web Store or Firefox Add-ons.

2

Navigate

Go to any website with a form — sign-up pages, checkout flows, surveys.

3

Click

Click the FillKit icon in the toolbar or use the keyboard shortcut.

4

Done

Every detected form field is filled with realistic, context-aware data.
